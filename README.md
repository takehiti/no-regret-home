"# no-regret-home

**後悔しない家づくりを、一度で実現するためのツール**

---

## これは何か

「家は二度建てないと分からない」

この言葉は、建築において広く知られた真実です。
しかし、一般の建築主にとって家を建てるチャンスは一度きりです。

**no-regret-home** は、この問題を解決するための判定・可視化ツール群です。

---

## 解決する問題

建築主が直面する以下の課題に対して、判断材料を提供します。

- **「それは無理です」の理由が分からない**
  → 数値・図・ビューで可視化し、なぜ無理か／どうすれば可能かを示す

- **専門知識がないと判断できない**
  → 専門用語を使わず、見て分かる形で提示する

- **完成後に気づく「こうしておけば…」**
  → 事前に視線・高さ・遮蔽などを検証し、納得して決める状態を作る

---

## 設計思想

### 1. **判定エンジンが核心**
UI非依存の純粋な計算・判定ロジックを中心に据えます。
これにより、Web / Unity / Revit など、あらゆるフロントエンドから利用可能です。

### 2. **ビューによる判断**
建築主にとって重要なのは「鑑賞用の3D」ではなく「判断できるビュー」です。
代表的な視点からの静止画・図を優先し、必要に応じて没入型体験へと進化させます。

### 3. **Revitはプロ向けツール**
一般ユーザーは Revit を持っていません。
Revitアドインは、設計者・検証者が使う「高精度チェックツール」として位置づけます。

### 4. **言語中立な設計**
エンジン層は特定の言語やプラットフォームに依存しない設計を目指します。
TypeScript / C# / Python など、必要に応じて実装できます。

## プロジェクト構成

```
no-regret-home/
├─ src/
│  ├─ engine/      # UI非依存の判定核（視線・高さ・法規など）
│  ├─ viewer/      # "判断できるビュー"生成（静止画・図）
│  └─ output/      # PDF/HTML などの説明資料
│
├─ tools/
│  ├─ revit/       # Revitアドイン（プロ向け検証）
│  └─ cli/         # CLIツール
│
├─ tests/          # テスト
└─ examples/       # 使用例・サンプルデータ
```

### 層構造

| 層 | 責務 | 独立性 |
|---|---|---|
| **engine** | 判定・計算ロジック | UI・3Dエンジン非依存 |
| **viewer** | ビュー生成 | engine を利用 |
| **output** | 資料生成 | viewer の結果を利用 |
| **tools** | プラットフォーム固有実装 | engine を利用する「例」 |

## MVP: 視線フェンス判定

最初の実装テーマは **「フェンスで視線を遮れるか？」** です。

### 入力
- 人の位置（複数可）
- 目線高さ
- フェンス位置（線）
- フェンス高さ
- 地盤レベル差（オプション）

### 出力
- 見える / 見えない の判定
- 視線ライン（どこを通って見えるか）
- 必要最小フェンス高さ
- 判断根拠（説明可能な情報）

### 出力形式
- 静止画ビュー（代表視点）
- 視線ラインと遮蔽ポイントが分かる図
- PDF レポート（将来）
